IDE - Visual Studio Code

Estrutura básica do HTML

<Voltar ao Início>

HTML, sigla para HyperText Markup Language (Linguagem de Marcação de Hipertexto), é a pedra fundamental da web. Ele consiste em uma estrutura hierárquica composta por elementos encapsulados em tags. Cada tag descreve o conteúdo ou a função de um elemento na página, como títulos, parágrafos, imagens e links. A estrutura básica do HTML geralmente inclui elementos como a tag que engloba todo o documento, as tags e que contêm metadados e conteúdo visível, respectivamente. As tags são combinadas para criar páginas web estruturadas e significativas.

Tags

<Voltar ao Início>

No HTML, as tags são elementos fundamentais que marcam e estruturam o conteúdo de uma página web. Elas funcionam como instruções que delimitam o início e o fim de diferentes tipos de elementos, como títulos, parágrafos, imagens e links. Cada tag é composta por um nome que é colocado entre sinais de menor e maior ("<" e ">"). Algumas tags podem conter atributos, que fornecem informações adicionais sobre o elemento, como seu comportamento ou aparência.

Para utilizar as tags em HTML, você as insere no código-fonte da página. A maioria das tags HTML possui uma forma de abertura e uma forma de fechamento. A forma de abertura inclui o nome da tag e, opcionalmente, atributos. A forma de fechamento inclui o mesmo nome de tag, mas precedido por uma barra ("/"). O conteúdo a ser afetado pela tag é inserido entre as formas de abertura e fechamento. Por exemplo, para criar um parágrafo, você usaria <p> para abrir e </p> para fechar, colocando o texto do parágrafo entre as duas tags: <p>Este é um exemplo de parágrafo.</p>. Assim, as tags desempenham um papel crucial na definição da estrutura e do significado do conteúdo em uma página web.

Textos

<Voltar ao Início>

As tags de texto em HTML desempenham um papel essencial na formatação e organização do conteúdo textual em uma página web. Elas permitem aos desenvolvedores controlar como o texto é exibido, desde a definição de títulos e parágrafos até a aplicação de ênfase e importância. As tags como <h1> a <h6> criam hierarquias de títulos, afetando o tamanho e a importância visual do texto, enquanto as tags <p> delimitam blocos de parágrafos. Além disso, as tags de formatação, como <em> e <strong>, permitem adicionar ênfase e importância semântica ao conteúdo. Combinadas e usadas com outras tags, as tags de texto oferecem a base para criar páginas web bem estruturadas e visualmente agradáveis.

Listas

<Voltar ao Início>

As tags de listas ordenadas e não ordenadas no HTML oferecem maneiras eficazes de apresentar informações de forma organizada. A tag <ul> é usada para criar listas não ordenadas, onde cada item é precedido por um marcador, como um ponto ou um símbolo. Por outro lado, a tag <ol> é empregada para criar listas ordenadas, onde cada item é numerado sequencialmente. Dentro dessas tags, os itens individuais são definidos usando a tag <li>. A combinação dessas tags permite aos desenvolvedores criar listas de itens, tópicos ou passos em tutoriais de maneira clara e compreensível para os usuários. As listas ordenadas e não ordenadas desempenham um papel importante na formatação de conteúdo e na facilitação da leitura em páginas web.

Links

<Voltar ao Início>

As tags de links, com destaque para a tag <a> em HTML, fornecem a essencial funcionalidade de navegação e conectividade na web. A tag <a>, ou âncora, é usada para criar links entre páginas, recursos ou seções de conteúdo. O atributo href dentro da tag especifica o destino do link, que pode ser uma URL externa, um arquivo local ou um identificador interno em uma página. Além disso, a tag permite a criação de links de texto ou imagens, tornando possível a navegação entre diferentes partes do conteúdo ou entre páginas distintas. Os links são uma parte essencial da experiência de navegação na web, permitindo aos usuários acessar rapidamente informações relevantes e explorar diferentes recursos online.

Como inspecionar elementos

<Voltar ao Início>

Inspecionar elementos HTML em um navegador é uma habilidade fundamental para iniciantes no desenvolvimento web. Essa ferramenta, amplamente disponível nos navegadores modernos, permite examinar e entender a estrutura, o layout e o estilo de uma página web em tempo real. Para começar, basta clicar com o botão direito do mouse em qualquer parte da página e selecionar a opção "Inspecionar" ou "Inspecionar elemento" no menu que aparece.

Isso abrirá uma janela na parte inferior ou lateral da janela do navegador, conhecida como "DevTools" ou "Ferramentas de Desenvolvedor". Dentro dessa interface, você verá uma representação do código HTML da página, onde cada elemento é exibido em uma hierarquia de árvore. Ao clicar nos elementos na árvore ou passar o mouse sobre eles na página, o elemento correspondente na página será destacado, permitindo que você veja como os elementos HTML estão estruturados e posicionados visualmente.

Além disso, a seção "Styles" ou "Estilos" na ferramenta de inspeção mostra os estilos CSS aplicados a cada elemento, o que é valioso para entender como a aparência da página é controlada. Você pode experimentar alterações diretamente na ferramenta para ver como diferentes estilos afetam o visual da página, embora essas mudanças não sejam persistentes após atualizar a página.

Em resumo, a inspeção de elementos HTML em um navegador é uma maneira poderosa de aprender sobre a estrutura e o estilo das páginas web, permitindo que iniciantes compreendam melhor como os sites são construídos e personalizados. Essa habilidade é especialmente útil para resolver problemas de layout, depurar erros de código e melhorar suas habilidades de desenvolvimento web.
